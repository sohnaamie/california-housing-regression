
# California Housing Price Prediction

This project uses multiple linear regression to predict house prices using the California Housing dataset.

## Features Used
- Median Income
- Average Rooms
- House Age
- Latitude
- Longitude

## Model Evaluation
- MAE: 0.54
- RMSE: 0.74

## Key Insights
- Income is the strongest predictor of house prices
- Location plays a significant role
- Removing highly correlated features improved interpretability

## Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn
